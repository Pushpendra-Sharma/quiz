
# Quiz App

A Quiz game based upon computer questions.


## Tech Stack

**Client:** React, Redux Toolkit, TailwindCSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/Pushpendra-Sharma/quiz.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
